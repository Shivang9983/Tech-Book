# 📚 Teach Book Store

Teach Book Store is a simple and responsive online bookstore website built using HTML, CSS, and JavaScript. It provides an attractive interface for users to browse books and explore the collection with a smooth user experience.

## ✨ Features

- 📖 Browse available books
- 🔍 Search functionality
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- ⚡ Fast and lightweight
- 🖼️ Interactive book showcase

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📁 Project Structure

```text
Teach-Book-Store/
│
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Shivang9983/Teach-Book.git
```

2. Open the project folder.

3. Run the project by opening `index.html` in your browser.
